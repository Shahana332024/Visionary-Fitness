Visionary Fitness - AI Powered Personal Trainer

Overview
Visionary Fitness is an AI-driven Full Stack application designed to track physical exercises using Computer Vision. It analyzes user posture in real-time to ensure correct form and count repetitions.

Tech Stack
- AI/CV:Python, OpenCV, MediaPipe
- Backend: Flask
- Frontend: HTML, CSS, JavaScript

 Features
- Real-time Tracking: Uses pose estimation to detect 33 key body landmarks.
- Exercise Analytics: Automatically counts repetitions for exercises like Jumping Jacks.
- Web Interface: Easy-to-use interface for starting workouts and viewing progress.
